Emoji Movie Guesser

A fun React & Tailwind CSS game where you guess famous movies and TV shows based on emojis!
______________________________________________________

Features

20 Levels: Supports both English & Arabic answers.

Smart Input: Forgives common spelling mistakes.

Flexible Play: Skip levels, track progress, and use Enter to submit.

Interactive: Visual feedback, animations, and a victory screen.
______________________________________________________

Quick Start

Create a React app: npx create-react-app emoji-game

Install Tailwind CSS.

Replace src/App.js with the EmojiGame.jsx code.

Run: npm start
______________________________________________________

How to Play

Look at the emojis > Type the movie/show name > Press Enter!

Built for a fun and smooth user experience.
